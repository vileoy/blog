Conda Local-Package-Auxiliary Installation Scheme

Installing Conda packages is really a headache problem, due to the strikingly slow Internet speed. Through tireless endeavor, I finally find a nice offline approach to get around it.

CLPAI Scheme Procedures

  1. Execute conda install xxx to install the desired package.
  2. When it raises “CondaHTTPError”, please copy the links into Manager to download those failed entries.
  3. Move the downloaded files to ~/usr/local/anaconda3/pkgs directory, and make sure that they possess consistent filenames.
  4. Next add the failed links to ~/usr/local/anaconda3/pkgs/urls.txt file.
  5. Finally rerun the conda install xxx command, and afterwards the package will be successfully installed.
